# Quotations

This is a little app for managing literary quotations.

## Server

### Running the server

`cd server` then `go run build` then `go run main.go`

### Server architecture

-   Go
-   SQLite
-   GORM
-   Gin

## Client

-   Preact
-   PWA
-   TypeScript

## Running the client

`cd client` then `npm i` then `npm run dev` or `npm run build` and `npm run preview`
